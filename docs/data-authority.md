# Data Authorities

All **Products** data MUST come from **Sanity.io**. No exceptions.

All **Cart** data MUST come from the **local Zustand store**. No exceptions.

All **Checkout** data MUST come from **local React state**. No exceptions.

All **Payments** data MUST come from the **PayTech API**. No exceptions.

All **Shipping** data MUST come from the **local shipping calculator**. No exceptions.

All **Messaging** data MUST come from the **WhatsApp Business API**. No exceptions.
